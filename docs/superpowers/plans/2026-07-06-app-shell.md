# App Shell (프론트오브하우스) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 단일 스크롤 프로토타입을 화면 라우터 기반 앱 셸(Title·Run·Settings·Summary·Pause)로 재구성한다 — 구조 먼저, 픽셀 아트는 이후 패스.

**Architecture:** 경량 바닐라 화면 라우터(`src/ui/router.cjs`)가 `.wrap` 안의 최상위 `.screen` 섹션을 토글한다. 기존 보드 마크업/`render()`/juice는 `run` 화면으로 **그대로** 감싼다(WebGL 스왑의 미래 seam). 신규 화면(title/settings/summary)은 mount 함수가 DOM을 구성한다. 라우팅 문자열은 전부 `t()` 로케일 키.

**Tech Stack:** 바닐라 JS, 런타임 의존성 0. `build.mjs`(esbuild 파싱게이트 + raw-concat)가 `src/`를 `prototype/index.html`로 빌드. 테스트 프레임워크 없음 → 검증 = `node build.mjs`(문법게이트) + `node tools/gate.cjs --fast`(밸런스 회귀 밴드 불변) + 수동 브라우저 QA.

**⚠️ raw-concat 제약 (build.mjs)**: 새 `src/ui/*.cjs`는 `main.cjs`가 whole-line `const { … } = require('./ui/x.cjs');`(후행주석 금지)로 **정확히 1회** require → resolveRequires가 본문을 인라인. 각 모듈은 `module.exports = { … };` 한 줄로 끝(빌드시 strip). ui 모듈끼리 서로 require 금지(공유 전역 스코프). 인라인 후 라우터 함수는 전역 — 화면 모듈이 런타임에 호출(정의 시점 아님)하므로 순서 무관(function 선언 사용).

**⚠️ 셸 딜리버리 범위**: 구조 + 최소 동작 화면. 시각 리치니스(픽셀 룩·리치 스탯)는 아트 A0/A1 후속. 스펙 대비 의도적 편차 1건: **⚡preview/⏩fast-fx 토글은 Run HUD에 유지**(인게임 컨트롤이라 플레이 중 접근이 옳음), Settings는 전역 프리퍼런스(오디오 뮤트·닉네임·reduced-motion)만. spec §3.4의 "토글 이관"은 이 근거로 조정.

## Task 3: src/ui/router.cjs — 화면 라우터

**Files:** Create `src/ui/router.cjs`

- [ ] **Step 1: 라우터 작성**

`src/ui/router.cjs` 생성 (function 선언 = 인라인 후 전역 호이스트; 화면끼리 require 없음):

```js
// src/ui/router.cjs — 앱 셸 화면 라우터. build.mjs가 main.cjs의 require로 인라인(전역 스코프).
// 화면 def = { el, mount?(el,ctx), unmount?() }. .wrap 안의 .screen 섹션을 토글.
const _SCREENS = {};
let _CUR = null;
function registerScreen(id, def){ _SCREENS[id] = def; }
function showScreen(id, ctx){
  const next = _SCREENS[id];
  if(!next) return;
  if(_CUR && _CUR !== id){
    const prev = _SCREENS[_CUR];
    if(prev){ if(prev.el) prev.el.classList.remove("active"); if(prev.unmount) prev.unmount(); }
  }
  if(next.el) next.el.classList.add("active");
  if(next.mount) next.mount(next.el, ctx);
  _CUR = id;
}
function currentScreen(){ return _CUR; }
module.exports = { registerScreen, showScreen, currentScreen };
```
