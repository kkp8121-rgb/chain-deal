## 검증된 발견 (3주장 전부 CONFIRMED)

### ① 옵션1 = 긴 체인 점수 보너스 → **효과 없음 (기각)**
| 시드뱅크 | 제어(OFF) | 옵션1(U:15/30/50) |
|---|---|---|
| 1..2000 | 대중 60% / 콤보 0.0% | 대중 **60%(무변)** / 콤보 0.1% |
| 2001..4000 | 대중 60% / 콤보 0.1% | 대중 **60%(무변)** / 콤보 0.1% |
| 4001..6000 | 대중 80% / 콤보 0.1% | 대중 **80%(무변)** / 콤보 0.1% |
| 8001..10000 | 대중 60% / 콤보 0.1% | 대중 **60%(무변)** / 콤보 0.1% |
| 20000·30000뱅크 | 60% | **60%(무변)** |

- **대중재미 패널을 단 한 번도 못 움직임** (opt1 panel ≡ control panel, 매 시드·매 곡선). 게이트 레버 아님.
- 8곡선(gentle 10/20/35 ~ steep 20/45/75, flat 8/18/32 ~ flatBig 12/26/44) **전부** 콤보 clear 0.0~0.3%(최대 only8:60=0.3%). 목표 0.5% 아래 고정.
- **부작용 = 부자감세**: 마스터리 13.6→최대 19.5%, 그리디 6.9→최대 10.4% 일관 버프. 밸런스 악화.
- caveat: "어떤 크기로도"를 무한 확장하면 붕괴급 보너스(flat +1000/chain)에서만 콤보가 오르나 명시 범위 밖 + 상위 페르소나를 더 버프 → 채택 근거 못 됨.

### ② 콤보러 0% clear의 진짜 원인 = **pick 정책(초반 자살)**, 성격 = **극단 봇 아티팩트**
- 죽는 판의 **97.9%가 maxChain ≤ 5**(체인 6 도달 전 사망). 3개 독립 시드뱅크서 97.2~97.9% 재현. → `rl≥6` 보너스는 **이미 살아남은 판에만 지급되는 死後 보상**이라 clear에 무력.
- **인과 격리**: shop을 cartel→balance→apex로 바꿔도 combo-pick clear 0.0~0.1%(shop 무효). **shop을 cartel로 고정**한 채 pick만 greedy로 바꾸면 0%→3.9~5.1%. 유일 변경 변수가 pick일 때만 clear 움직임.
- 콤보러 avgAnte ≈ 1.7 → **안테 1~2에서 대부분 즉사.** 체인폭발만 노려 초반 생존을 자살하는 **극단 전략 봇**.
- **판단**: 0% clear는 "고쳐야 할 결함"이 아니라 극단 플레이스타일의 자연 귀결. 같은 shop에서 greedy가 4~7% 내는 것이 게임 자체는 클리어 가능함을 증명.
- ⚠️ **함정**: 콤보러 fun 7.1~7.4(최고 재미)인데 clear ~0%. **정량 재미가 승률을 심하게 과대평가.** "재밌다"와 "이긴다"가 분리된 페르소나 → 콤보러 clear를 P2 성공지표로 쓰면 안 됨.

### ③ 초반 완화(relief) → 방향은 맞으나 **비강건·비단조 + 난이도 연화**
- 캐주얼 fun을 6.0 임계 **부근**까지만 올림(대부분 여전히 5.4~6.0).
- **비강건·비단조**: 80% 도달은 특정 시드뱅크의 단일 relief값에서만(A뱅크 ×0.85 / B뱅크 ×0.80), 인접값·타 시드뱅크선 60%로 붕괴. **control 패널 자체가 시드뱅크마다 60%/80%로 흔들림** — 캐주얼이 6.0 칼날 위라 대중재미 지표 전체가 현 밸런스에서 취약.
- 콤보 clear는 relief 무관 ~0% 고정.
- **난이도 연화 부작용**: 마스터 12→최대 25%, 그리디 6→최대 17% 완화강도 비례 단조 상승.

### ★ 신규 사각지대 — **스릴러는 어떤 레버로도 안 움직임**
스릴러 fun = 모든 레버(opt1·relief·8곡선·전 시드)에서 **5.85~5.91 완전 고정**. 스릴러 flow는 ~0.9로 높은데 fun이 낮음 → tension/dopamine 축 문제 의심. **현 레버 풀엔 스릴러를 건드리는 레버가 존재하지 않는다.** 하위 2페르소나(스릴러·캐주얼)가 fun<6이라 3/5=60%에 갇힘 — 캐주얼만 겨우 움직이는 현 레버로 80%는 구조적으로 불안정.

## 개발 세션 다음 액션 (우선순위·검증법)

> 파라미터·코드는 개발 몫. QA는 방향과 재QA 성공기준만 넘긴다.

**A. [최우선] 스릴러·캐주얼 fun 원인 분해 → 겨냥 레버 신설**
- 점수/체인 보너스 계열 **중단**. `p2-diagnose.cjs`류를 확장해 스릴러·캐주얼이 왜 fun<6인지 **5축 성분 분해**(스릴러=tension/dopamine 의심, flow는 이미 높음). 원인 성분을 겨냥한 레버(초반 리스크-리워드 변동성, 하이롤 dopamine 스파이크 등)를 그레이박스로 격리 구현.
- **재QA 성공기준**: 신규 3시드뱅크(SEED_OFFSET 40000/50000/60000, N≥2000)에서 스릴러 또는 캐주얼 fun≥6이 **3뱅크 전부** 성립 → 그때만 panel 80% 강건성 인정.

**B. [차순위] 게이트 재정의 + 콤보러 clear KPI 제외**
- 단일 시드 우연 80%를 PASS로 세지 말 것. 게이트 = **독립 3시드뱅크 전부 ≥80%(비단조 붕괴 없음)**.
- 콤보러는 **fun-only 평가**(fun≥6 이미 PASS), clear는 비지표.

**C. [보조] relief 단독 채택 금지**
- A 레버가 스릴러를 6 위로 올린 뒤, 캐주얼 마진 확보용으로만 소량 병행. relief 강도는 마스터/그리디 clear 상승폭 상한(control 대비 +5pp 이내)으로 묶어 검증.
