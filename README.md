## 룰 (한 문장)
손패 3장 중 하나를 골라 줄에 깐다 — 바로 왼쪽 카드와 **같은 무늬 · 같은 숫자 · ±1 연속**이면 연결, 길게 이을수록 **배율 폭발**. 8장 깔아 목표 넘기면 통과 → 상점에서 덱 강화. **안테 8 보스 격파 = 승리.**

## 핵심 메커니즘
- **연결 = 포커 직관**: 플러시(같은 무늬) / 페어(같은 숫자) / 스트레이트(연속)
- **체인 배율**: 연속 연결이 길수록 점수 폭발 (수읽기 = "다음 다리"를 생각하며 카드 선택)
- **덱빌딩**: 블라인드마다 상점 — 덱 압축 / 카드 효과(와일드·배율석·황금) / 손패확장·리롤 스킬 / 부적
- **보스 블라인드 5종**: 특수 규칙으로 전략 방해 → 적응 강요
- **포커 족보 보너스**: 8장 줄로 텍사스 홀덤 족보 인식 → "체인 길게" vs "플러시·포카드 세팅"의 두 번째 전략축
- **연출**: 정적 카드 + 트윈/파티클/화면연출 + WebAudio 절차 사운드 ("손맛")
